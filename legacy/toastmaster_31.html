Import Roles HTML
Reads the role history table and assigns roles to existing or newly created meeting records
How it works: Each row maps a member + date + role. If the meeting date doesn't exist it is created as a Regular meeting. Duplicate role assignments for the same meeting are ignored (first one wins).
Import Meeting CSV
Review parsed meetings before importing — duplicates are detected automatically
Expected columns: Date, Theme, Special Meeting, Notes, Guests, Toastmaster, General Evaluator, Humour Corner, Ah Counter, Grammarian, Timer, Table Topic Master, Table Topic Evaluator Odd, Table Topic Evaluator Even, Speaker 1–6, Evaluator 1–3, Backup Speaker, Chair, SAA, Attendee, Apologies — Guests, Attendee and Apologies may be multi-line quoted fields.
Import Pathways HTML
Reads the speech history table from a FreeToastHost-style HTML export and merges into member records
How it works: The importer finds the speech history table (columns: Name, Date, Allocated Duration, Title, Manual / Project, Introduction), matches each row to a member and pathway, then merges completions without overwriting existing data.
Import Pathways CSV
Review parsed pathway data before merging into member records
Expected CSV columns: Member  |  Pathway  |  Notes   — First row must be the header. Existing pathway data is preserved (merge only adds new entries).
